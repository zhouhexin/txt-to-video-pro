# 🎙️ 为已生成视频添加配音 - 功能说明

**更新日期**: 2026-04-24  
**版本**: v1.3

---

## 🎯 功能概述

现在可以**为任何已生成的视频添加配音**，无需重新生成视频！

### 核心能力

1. ✅ **随时添加** - 为任何已完成的任务添加配音
2. ✅ **重新生成** - 可以更换音色重新生成配音
3. ✅ **即时预览** - 生成后立即播放完整视频
4. ✅ **灵活修改** - 不满意可以随时重新生成

## 🔧 技术实现

### 组件结构

```
Showcase.vue (成果展示)
  └─ VoiceoverConfig.vue (配音配置)
       ├─ 音色选择
       ├─ 生成配音
       ├─ 配音预览
       └─ 合并到视频
```

### 数据流

```typescript
// 1. 生成配音
generateAllAudios({
  task_id: taskId,
  script_id: scriptId,
  voice_id: 'xiaoxiao'
})
  ↓
// 2. 获取配音列表
getTaskAudios(taskId)
  ↓
// 3. 显示预览
voiceovers.value = [...]
  ↓
// 4. 合并到视频
mergeAudioVideo({
  task_id: taskId
})
  ↓
// 5. 刷新视频列表
axios.get(`/api/v1/videos/${taskId}`)
```

### API 调用

```typescript
// 批量生成配音
POST /api/v1/audios/generate-all
{
  "task_id": "task_xxx",
  "script_id": 1,
  "voice_id": "xiaoxiao"
}

// 获取配音列表
GET /api/v1/audios/:task_id

// 合并配音到视频
POST /api/v1/audios/merge
{
  "task_id": "task_xxx"
}
```

### 配音技巧

1. **文本准备**
   - 系统自动使用分镜的"画面描述"
   - 确保描述清晰、流畅
   - 适当添加标点符号

2. **音色匹配**
   - 根据视频内容选择
   - 考虑目标受众
   - 保持风格一致

3. **音量平衡**
   - 配音：100%
   - BGM：20-30%
   - 音效：30-50%

4. **试听验证**
   - 生成后先试听
   - 检查语速、停顿
   - 确认发音准确

## ❓ 常见问题

### Q1: 可以为多久以前的视频添加配音？

**A**: 任意时间！
- 只要任务还在数据库中
- 无论何时生成的视频
- 都可以随时添加配音

### Q2: 重新生成配音会覆盖旧的吗？

**A**: 是的。
- 新配音会覆盖同一任务的旧配音
- 建议先试听再合并
- 合并前可以多次重新生成

### Q3: 生成失败怎么办？

**A**: 可以重试。
- 检查网络连接
- 确认文本不为空
- 点击重新生成即可

### Q4: 配音可以下载吗？

**A**: 当前版本：
- ✅ 可以下载完整视频（含配音）
- ⚠️ 暂不支持单独下载配音音频

### Q5: 可以只生成部分分镜的配音吗？

**A**: 当前版本：
- ❌ 暂不支持部分生成
- ✅ 一次性生成所有分镜
- 💡 如需部分，可手动编辑分镜描述

### Q6: 生成配音需要多长时间？

**A**: 
- 约 2-5 秒/100 字
- 5 个分镜（1000 字）约 1-2 分钟
- 网络状况会影响速度

### Q7: 配音收费吗？

**A**: 
- ✅ 完全免费（Edge TTS）
- ✅ 无使用次数限制
- ✅ 无时长限制

```vue
<template>
  <VoiceoverConfig
    :task-id="taskId"
    :script-id="scriptId"
    :total-shots="totalShots"
    @complete="handleComplete"
    @merged="handleMerged"
  />
</template>

<script setup>
import VoiceoverConfig from '@/components/VoiceoverConfig.vue'

const handleComplete = (voiceovers) => {
  console.log('配音生成完成', voiceovers)
}

const handleMerged = (videoUrl) => {
  console.log('配音已合并', videoUrl)
  // 刷新视频列表
}
</script>
```

### 状态管理

```typescript
// 配音状态
interface VoiceoverState {
  generating: boolean    // 生成中
  merging: boolean       // 合并中

  totalCount: number     // 总数量

}
```
