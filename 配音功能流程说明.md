# 🎙️ 配音功能流程说明

**更新日期**: 2026-04-24  
**版本**: v1.2

---

## 📋 新流程概览

配音功能已移至**视频生成的最后一步**，在成果展示页面添加。

### 完整流程

```
1. 智能剧本 → 生成剧本
       ↓
2. 分镜生成 → 生成分镜图
       ↓
3. 视频生成 → 生成视频片段 + 配置 BGM/音效
       ↓
4. 合并视频 → 仅合并视频（不含配音）
       ↓
5. 成果展示 → 🎙️ 添加配音（新增步骤）
       ↓
6. 最终导出 → 完整视频（含配音+BGM+ 音效）
```

---

## 🎯 为什么调整流程？

### 原流程问题

1. **等待时间长**: 配音生成需要时间，用户需要等待
2. **无法预览**: 合并前无法听到配音效果
3. **修改困难**: 配音后发现问题需要重新生成

### 新流程优势

1. ✅ **快速合并**: 先生成视频，快速看到效果
2. ✅ **灵活配置**: 可以随时调整配音音色
3. ✅ **即时预览**: 生成后立即播放完整视频
4. ✅ **可重复生成**: 不满意可以重新生成配音

## 📖 使用指南

### Step 1: 视频生成页面

**位置**: `/video`

**操作**:
1. 配置视频参数（分辨率、时长等）
2. 配置 BGM 和音效（可选）
   - ⚠️ 配音配置已隐藏
3. 生成所有视频片段
4. 点击"合并视频"

**结果**: 生成不含配音的完整视频

---

### Step 2: 成果展示页面（新增）

**位置**: `/showcase`

**操作**:
1. 找到刚生成的任务
2. 点击"查看成果"
3. 在详情页面找到"🎙️ 添加配音"模块
4. 选择音色（8 种可选）
5. 点击"生成所有分镜配音"
6. 等待生成完成（约 2-5 秒/100 字）
7. 预览配音效果
8. 点击"合并配音到视频"

**结果**: 生成含配音的完整视频

## 🔧 技术实现

### 新增组件

**文件**: `frontend/src/components/VoiceoverConfig.vue`

**功能**:
- 音色选择（8 种）
- 批量生成配音
- 进度显示
- 配音预览

**API 调用**:
```typescript
// 生成所有配音
generateAllAudios({
  task_id: taskId,
  script_id: scriptId,
  voice_id: 'xiaoxiao'
})

// 获取配音列表
getTaskAudios(taskId)
```

### 修改组件

**文件**: `frontend/src/views/VideoGen.vue`

**变更**:
- 移除合并时的自动配音生成
- 添加提示说明配音在成果页面添加
- AudioConfig 默认禁用配音

**文件**: `frontend/src/components/AudioConfig.vue`

**变更**:
- `enableVoice` 默认值改为 `false`
- 配音配置 UI 保留但默认隐藏

## ❓ 常见问题

### Q1: 为什么配音移到最后？

**A**: 
- 让用户先快速看到视频效果
- 可以灵活调整配音音色
- 减少等待时间

### Q2: 可以跳过配音吗？

**A**: 可以！
- 配音是可选功能
- 直接下载不含配音的版本

### Q3: 配音不满意可以重新生成吗？

**A**: 可以！
- 随时重新选择音色
- 重新生成配音
- 覆盖之前的版本

### Q4: 生成配音需要多长时间？

**A**: 
- 约 2-5 秒/100 字
- 5 个分镜（1000 字）约 1-2 分钟

### Q5: 配音收费吗？

**A**: 
- ✅ 完全免费（Edge TTS）
- 无使用限制
