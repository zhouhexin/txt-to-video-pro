## 🎙️ 配音功能详解

### 什么是 AI 配音？

AI 配音使用微软 Edge TTS 技术，将分镜描述文字转换为自然的语音旁白。

### 可用音色（8 种）

| 音色 ID | 音色名称 | 适用场景 |
|--------|----------|----------|
| **xiaoxiao** | 温柔女声 | 新闻、有声书、文旅宣传 ⭐ |
| **xiaoyi** | 活泼女声 | 卡通、动画、轻松内容 |
| **yunjian** | 激情男声 | 体育、励志、宣传片 |
| **yunxi** | 阳光男声 | 有声书、解说、vlog |
| **yunxia** | 可爱男声 | 卡通、儿童内容 |
| **yunyang** | 专业男声 | 新闻、纪录片 |
| **xiaobei** | 东北话 | 方言视频、喜剧 🎭 |
| **xiaoni** | 陕西话 | 方言视频、地方特色 🎭 |

### 配置步骤

1. **启用配音**: 打开"启用配音"开关

2. **选择音色**: 从下拉列表选择合适的音色
   - 文旅宣传 → 推荐 `xiaoxiao`（温柔女声）
   - 纪录片 → 推荐 `yunyang`（专业男声）
   - 喜剧视频 → 推荐 `xiaobei`（东北话）

3. **生成配音**: 
   - 系统会自动使用分镜的"画面描述"作为配音文本
   - 点击"合并视频"时自动生成

### 试听功能

⚠️ **注意**: Edge TTS 需要实时生成，暂不支持在线试听。

您可以在生成配音后，在"成果展示"页面播放完整视频来试听效果。

### 生成时间

| 文本长度 | 预计时间 |
|----------|----------|
| 50 字 | ~2 秒 |
| 100 字 | ~3 秒 |
| 200 字 | ~5 秒 |

### 成本

## 🎵 BGM 功能详解

### 什么是 BGM？

BGM（Background Music）即背景音乐，为视频营造氛围。

### 可用 BGM（10 首）

| ID | 名称 | 分类 | 风格 | 时长 |
|----|------|------|------|------|
| bgm_001 | 清晨阳光 | 自然 | 轻松 | 3:00 |
| bgm_002 | 古风韵 | 古风 | 优雅 | 4:00 |
| bgm_003 | 城市脉搏 | 现代 | 活力 | 3:20 |
| bgm_004 | 宁静夜晚 | 自然 | 安静 | 3:40 |
| bgm_005 | 山水之间 | 自然 | 悠扬 | 4:20 |
| bgm_006 | 欢乐时光 | 生活 | 快乐 | 3:10 |
| bgm_007 | 史诗征程 | 史诗 | 激昂 | 5:00 |
| bgm_008 | 田园风光 | 自然 | 悠闲 | 3:30 |
| bgm_009 | 科技感 | 现代 | 未来 | 4:00 |
| bgm_010 | 温馨家园 | 生活 | 温暖 | 3:20 |

### 配置步骤

1. **启用 BGM**: 打开"启用 BGM"开关

2. **选择 BGM**: 
   - 按分类筛选：自然、古风、现代、生活、史诗
   - 按风格筛选：轻松、优雅、活力、安静等
   - 鼠标悬停可查看风格标签

3. **调节音量**: 
   - 默认 30%（推荐）
   - 范围：0% - 100%
   - 建议：配音时 BGM 音量 20-30%，纯音乐时 50-70%

4. **试听**: 
   - 点击"🔊 试听"按钮
   - 实时播放 BGM 片段
   - 再次点击停止播放

### 搭配建议

| 视频类型 | 推荐 BGM | 音量 |
|----------|----------|------|
| 文旅宣传 | 清晨阳光、山水之间 | 25% |
| 古风视频 | 古风韵 | 30% |
| 城市宣传片 | 城市脉搏 | 30% |
| 纪录片 | 宁静夜晚、田园风光 | 20% |

## 🔊 音效功能详解

### 什么是音效？

音效（Sound Effects）是增强视频真实感的声音元素，如鸟鸣、流水、风声等。

### 可用音效（12 个）

| ID | 名称 | 分类 | 时长 |
|----|------|------|------|
| sfx_001 | 鸟鸣声 | 自然 | 2 秒 |
| sfx_002 | 流水声 | 自然 | 5 秒 |
| sfx_003 | 风声 | 自然 | 3 秒 |
| sfx_004 | 人群嘈杂 | 环境 | 4 秒 |
| sfx_005 | 雨声 | 自然 | 3 秒 |
| sfx_006 | 雷声 | 自然 | 2 秒 |
| sfx_007 | 海浪声 | 自然 | 5 秒 |
| sfx_008 | 火焰燃烧 | 自然 | 3 秒 |
| sfx_009 | 脚步声 | 人物 | 1 秒 |
| sfx_010 | 笑声 | 人物 | 2 秒 |
| sfx_011 | 钟声 | 建筑 | 3 秒 |
| sfx_012 | 车辆驶过 | 城市 | 2 秒 |

### 配置步骤

1. **启用音效**: 打开"启用音效"开关

2. **选择模式**:
   - **自动推荐**（推荐）: 系统根据分镜描述自动推荐
   - **手动选择**: 从列表手动挑选音效

3. **自动推荐**:
   - 开启"自动推荐"开关
   - 系统分析分镜描述中的关键词
   - 自动匹配合适的音效

4. **手动选择**:
   - 关闭"自动推荐"
   - 从下拉列表选择音效（可多选）

### 智能推荐示例

| 场景描述 | 推荐音效 |
|----------|----------|
| "清晨的湖边有鸟鸣" | 鸟鸣声、流水声 |
| "微风吹过树林" | 风声、鸟鸣声 |
| "下雨天的街道" | 雨声、车辆驶过 |
| "热闹的集市" | 人群嘈杂、笑声 |
| "海边的黄昏" | 海浪声、风声 |

### 搭配建议

- ✅ **少而精**: 每个场景 1-2 个音效即可
- ✅ **音量适中**: 音效不要盖过配音
- ✅ **自然过渡**: 避免音效突兀

## ❓ 常见问题

### Q1: 配音听起来不自然怎么办？

**A**: 尝试以下方法：
1. 更换音色（不同音色适合不同内容）
2. 调整文本（避免过长句子）
3. 添加标点符号（帮助断句）

### Q2: BGM 声音太大/太小怎么办？

**A**: 
1. 在配置页面调节 BGM 音量滑块
2. 推荐音量：
   - 有配音：20-30%
   - 纯音乐：50-70%

### Q3: 音效推荐不准确怎么办？

**A**:
1. 关闭"自动推荐"
2. 手动选择合适的音效
3. 或者修改分镜描述，使其更具体

### Q4: 合并视频失败怎么办？

**A**: 检查以下几点：
1. 所有视频片段是否生成完成
2. ffmpeg 是否正确安装
3. 查看后端日志错误信息

### Q5: 可以只使用 BGM，不用配音吗？

**A**: 可以！
1. 关闭"启用配音"开关
2. 只启用 BGM 即可
3. 系统会生成纯音乐视频

### Q6: 可以自定义配音文本吗？

**A**: 当前版本使用分镜的"画面描述"作为配音文本。

**变通方法**:
1. 在生成剧本时，在"画面描述"中写入想要的旁白
2. 或联系开发者添加自定义旁白功能

### Q7: 生成的音频文件在哪里？

**A**: 
```
output_tasks/{task_id}/audios/
├── shot_0_voice.wav
├── shot_1_voice.wav
└── ...
```

### Q8: 可以替换 BGM 文件吗？

**A**: 可以！

**步骤**:
1. 准备 MP3 文件
2. 放入 `backend/bgm_library/` 目录
3. 编辑 `backend/bgm_library/bgm_library.json`
4. 重启后端服务

## 💡 最佳实践

### 配音技巧

1. **选择合适音色**
   - 文旅宣传 → xiaoxiao（温柔女声）
   - 纪录片 → yunyang（专业男声）
   - 喜剧 → xiaobei（东北话）

2. **文本优化**
   - 每句 50-100 字为宜
   - 添加适当标点
   - 避免生僻字

3. **音量平衡**
   - 配音音量：100%
   - BGM 音量：20-30%
   - 音效音量：30-50%

### BGM 搭配

1. **情绪匹配**
   - 快乐场景 → 欢乐时光
   - 悲伤场景 → 宁静夜晚
   - 宏大场景 → 史诗征程

2. **风格统一**
   - 古风视频 → 古风韵
   - 现代城市 → 城市脉搏
   - 自然风光 → 山水之间

3. **时长考虑**
   - 短视频（<1 分钟）→ 截取高潮部分
   - 长视频 → 选择循环性好的音乐

### 音效使用

1. **画外音原则**
   - 看到鸟 → 加鸟鸣
   - 看到水 → 加流水
   - 看到雨 → 加雨声

2. **层次分明**
   - 主要音效：突出显示
   - 背景音效：若有若无
   - 避免过多：1-2 个/场景

3. **自然过渡**
   - 淡入淡出
   - 避免突兀
   - 与环境融合

### 完整案例

**案例**: 文旅宣传片《西安大唐芙蓉园》

```
配置:
- 配音：xiaoxiao（温柔女声）
- BGM: 古风韵（30% 音量）
- 音效：自动推荐

分镜 1: 开场全景
  - 画面：大唐芙蓉园夜景
  - 配音："夜幕降临，大唐芙蓉园灯火辉煌..."
  - BGM: 古风韵（前奏）
  - 音效：人群嘈杂（远处）

分镜 2: 建筑特写
  - 画面：亭台楼阁
  - 配音："精美的建筑，展现了盛唐风貌..."
  - BGM: 古风韵（主旋律）
  - 音效：风声（轻微）

分镜 3: 湖面倒影
  - 画面：湖水中的倒影
  - 配音："湖水如镜，倒映着千年的繁华..."
  - BGM: 古风韵（高潮）

```
